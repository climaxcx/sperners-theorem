# Sperner's Theorem in Lean

This project is an independent formalization of Sperner's theorem in Lean 4 using Mathlib.

The proof follows the presentation from the University of Oxford course C8.3 Combinatorics, which uses the LYM inequality. While a shorter probabilistic proof is also present in the notes, its implementation in Lean seems more difficult. The proof method chosen also uses more of the Finset methods, which is also why it was preferred, as the main goal of the project is educational.
